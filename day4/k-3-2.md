# 身長(cm)と体重(kg)を入力したら、BMIの値を計算してくれるプログラムを作成してみよう。

## 私
```
# 変数定義
height = int(input("身長"))/100
weight = int(input("体重"))
bmi = weight / height ** 2

# 出力
print(bmi)
```

## 先生
```
#表示　print
print("BMIを計算します")

#入力　input → 文字列
cm = int( input("身長：") )
kg = int( input("体重：") )

#データ型の変換
cm = int(cm)
kg = int(kg)

#計算
m = cm /100
bmi = kg / (m*m) #m**2

#表示
print("BMIは",int(bmi),"です。")
```
