# 練習２
# アトラクションチェッカーアプリを作ってみましょう。<br>
「身長を入力して、１３０cm未満だった場合アトラクションには乗れませんとお知らせしてくれる」

## 私
```
# 変数定義
height = int(input("身長(cm)を入力してください。"))

# 処理
if height < 130:
    print("ジェットコースターには乗れません！")
    print("大きくなったらまた来てね！")
else:
    print("たのしんでいってらっしゃ～い！")
```

## 先生
```
#身長を入力
cm = input("身長cmを入力；")

#条件分岐
if cm < 130:
    print("のれないよ～")
    print("またきてね")

else:
    print("たのしんでね！")
```
