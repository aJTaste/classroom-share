# 数あてゲームを作成してみましょう！<br>
条件は以下の通りです！
- ①ファイル名は「k-4」としてください。
- ②答えとなる値はあらかじめソースコード内にいれておいてください！
- ③値を入力し、同じ値の場合は「正解！」と表示　違う場合は「不正解！」
- ④プログラムの終了時には、「また挑戦してください！」と表示

## 私
```
# 変数定義
value = int(32)
answer = int(input("値を入力してください。"))

# 処理
if answer == value:
    print("正解！")
else:
    print("不正解！")

print("また挑戦してください！")

```
